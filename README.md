# UrbanThreads – Full Stack E-Commerce Website

UrbanThreads is a full-stack e-commerce application built using modern technologies.

## Tech Stack

Frontend
- React
- Tailwind CSS
- Axios

Backend
- Spring Boot
- Spring Security
- JWT Authentication
- MySQL

Payment Integration
- Razorpay

Authentication
- Email & Password Login
- Google OAuth Login

## Features

- User authentication
- Google login
- Product browsing
- Add to cart
- Order checkout
- Razorpay payment integration
- Order history
- Secure JWT authentication

## Run Locally

### Backend


cd backend
mvn spring-boot:run


### Frontend


npm install
npm run dev


Frontend runs on:

http://localhost:5174


Backend runs on:

http://localhost:8080
